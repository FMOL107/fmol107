---
title: Extraer contraseñas de un perfil de Firefox
---

https://github.com/unode/firefox_decrypt
https://github.com/lclevy/firepwd

Para extraer las contraseñas de un perfil de firefox necesitamos dos ficheros:
- **logins.json**: Contiene las entradas de las contraseñas (sitios web, nombres de usuario, contraseñas cifradas).
- **key4.db**: Contiene las claves maestras cifradas utilizadas para descifrar las contraseñas.
